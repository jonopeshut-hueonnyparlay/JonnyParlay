# KairosEdge Results

One folder per game date: `YYYY-MM-DD/`. See `projects/_RESULTS_STANDARD.md` for the template.

Kairos-specific reminders for `slate.md`:
- Game, halftime score, deficit
- Base-rate comeback % from `comeback-probability-model`
- Live full-game winner price across all books at HT
- Book chosen + entry price
- Any RLM signal at the time of entry

`outcome.md` should include the **closing price on the trailing team** so you can compute the CLV captured per trade — that's the only reliable Kairos edge signal short-term.
